# 📚 Задания по C#

---

## 📌 Задание №1
Описание задания:
Создать программу для учета книг в библиотеке. БД должна включать
сущности «Книги» (Books), «Читатели» (Readers), «Выданные книги»
(IssBooks), «Возвращенные книги» (RetBooks). Обеспечить возможность добавления книг и читателей, учета выдачи и возврата книг, получения информации
о книге и о читателе


### 🔧 Функциональность

Программа позволяет:

- ➕ Добавлять книги
- 👤 Добавлять читателей
- 📖 Выдавать книги читателям
- 🔄 Возвращать книги
- 🔍 Просматривать информацию:
  - о книге
  - о читателе

---

## 🏗️ Структура первого задания

### 📘 `Book`
Класс книги:

- Название
- Автор
- Год издания
- Количество страниц
- Жанр
- Язык
- Статус (выдана или нет)

---

### 👤 `Reader`
Класс читателя:

- ФИО
- Возраст
- Телефон
- Email
- Список взятых книг

---

### 📥 `IssBooks`
Информация о выданных книгах:

- Книга
- Читатель
- Дата выдачи

---

### 📤 `RetBooks`
Информация о возвращённых книгах:

- Книга
- Читатель
- Дата возврата

---

### 🏛️ `Library`
Основной класс управления:

- Хранение книг (`Dictionary`)
- Хранение читателей (`Dictionary`)
- История выдачи (`List`)
- История возврата (`List`)

---

## ▶️ Запуск программы

1. Открыть проект в Visual Studio Code  
2. Собрать проект  
3. Запустить программу  

---

## 💻 Интерфейс

Программа работает через консольное меню:

```
1 - Добавить книгу
2 - Добавить читателя
3 - Выдать книгу
4 - Вернуть книгу
5 - Информация о книге
6 - Информация о читателе
0 - Выход
```
